# Buchat App

Buchat, kullanıcıların güvenli ve gizlilik odaklı şekilde sohbet etmelerini sağlayan bir mobil uygulamadır.

## Özellikler
- Gerçek zamanlı mesajlaşma
- Güçlü gizlilik ve güvenlik
- Kullanıcı dostu arayüz
